튜토리얼하고 진짜 내 것인줄 아는 사람이 되지 않으려 노력 중입니다.

기술명 로고 활용 수준
Vue.js
Vue를 활용하여 Nodejs 백단과 연결하여 라벨링툴을 만들어보았으며 vuex나 mysql, mongodb 등을 활용하여 적절한 통신 요청, 응답을 처리할 수 있습니다.
Node.js
Express를 활용하여 웹 API 서버를 만들 수 있으며 react, vue와 결합하여 서비스를 구성할 수 잇습니다. 또한 다양한 npm 패키지를 활용하여 적절한 동작, 통신 요청, 응답을 알맞게 처리할 수 있습니다.
JavaScript
ES6 문법을 활용하여 js 언어를 사용하는 라이브러리, 기술을 사용할 수 있습니다. 기본적인 문법은 물론 다양한 토이프로젝트를 진행하며 처음보는 코드도 이해하여 응용할 수 있습니다.
C++
Python
java
Html+Css
다음의 언어들의 기본 문법을 다룰 수 있으며 작성된 코드를 보며 원리를 이해하고 알맞게 응용할 수 있습니다.

궁금하고 필요해서 공부해 봤어요.

아래의 기술들을 사용해본 경험이 있습니다.

기술명 로고 활용 수준
Flask
텐서플로를 다루며 여러 토이프로젝트에 사용했으며 간단한 API 서버 구현 및 AI 추론 서버로 사용할 수 있습니다.
Aws
aws s3 서비스를 활용하여 정적 웹 호스팅을 해보았고 웹 서비스의 이미지 제공용으로 사용해보았습니다.
MongoDB
기본적인 mongodb 문법을 사용할 수 있으며 회원정보, 소켓과 결합한 실시간 데이터 갱신 등에 활용 해보았습니다.
MySQL
기본적인 SQL 문법을 다룰 수 있으며 웹과 연동하여 회원 정보, 세션 정보를 저장하는 용도로 사용해보았습니다.
Tensorflow
기본적인 텐서플로를 이해하고 있으며 모델을 작성할 수 있습니다. 복잡한 응용까지는 못하나 관련 자료, 논문을 보며 이해하고 구현할 수 있도록 노력하고 있습니다. 여러 토이프로젝트를 진행하며 텍스트, 이미지 분야에 대한 이해를 넓히고 있습니다.
Docker
도커의 기본적인 사용방법을 다룰 수 있으며 여러 컨테이너들을 엮는 docker-compose 를 사용하여 개발 환경을 구성할 수 있습니다.
